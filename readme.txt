1. Installation
virtaulenv:
# virtualenv newenv
# source newenv/bin/activate
# pip install -r requirements.txt

create credentials.json in the project root directory and fill the data

{
  "db_host": "127.0.0.1",
  "db_scheme": "",
  "db_user": "",
  "db_password": ""
}

2. Run the simulator:
# source venv/bin/activate
# python simulator.py
